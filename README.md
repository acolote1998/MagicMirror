# Info Screen

This project is designed to create a dynamic information display for your living room screen. It fetches real-time data from various sources, including public transportation services and weather, to provide relevant information at a glance. The information displayed includes upcoming departures for trains, buses, and the Tunnelbana (subway) near your location, along with the current weather, weekday, and date.

Features
Real-time public transportation updates: Fetches and displays upcoming departures for nearby trains, buses, and Tunnelbana services.
Weather information: Displays current weather conditions, such as temperature and weather description.
Date and Weekday: Displays the current date and day of the week.
Customization: The project fetches data using API keys, allowing you to tailor it to your specific location and preferences.
How it works
The application uses several APIs to fetch data:

SL Lokal Trafik API: Provides real-time data for public transportation services (trains, buses, Tunnelbana).
Weather API: Retrieves current weather information for your location.
Date and Weekday: The current date and weekday are dynamically fetched from the system.
The data is then displayed on your screen in a user-friendly and easy-to-read format, making it perfect for a living room display or any other information board setup.
